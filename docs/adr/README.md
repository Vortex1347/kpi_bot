# ADR Index

Architecture Decision Records (ADR) для проекта.

## Status Legend
- `accepted` — решение принято и активно.
- `superseded` — заменено новым ADR.
- `proposed` — обсуждается.

## How To Add New ADR
1. Создать файл `NNNN-short-title.md`.
2. Описать `Context`, `Decision`, `Consequences`.
3. Добавить ссылку в этот индекс.
4. Обновить `ARCHITECTURE.md`/`PROJECT.md`, если решение влияет на них.
